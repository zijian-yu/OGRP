   Blast is a search tool based on local sequence alignment algorithms. It can compare the input nucleic acid or protein sequence with known sequences in the database, and obtain sequence similarity and other information. Select the Blast subroutine: blastp or blastn. Upload target file and query file. The default parameters are E value<1e-5, score>100, and identity value<60.
Example : prot.fasta  prot2.fasta  nucl.fa
