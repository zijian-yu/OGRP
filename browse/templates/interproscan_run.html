Gene Function Search

  Interproscan is a gene annotation tool that allows the annotation of multiple databases in a single run.
  We analyzed 912,552 genes from 25 Fabaceae for annotation and obtained a total of 8,470,419 annotations. We have piggybacked on the platform so that researchers can not only easily access but also search to find annotation information of gene id, sequence length, analysis software, gene annotation information, start and stop positions, IPR number and Go number. In addition, we performed statistics on the structural domains of each of the 25 Fabaceae and outgroup grape species.
