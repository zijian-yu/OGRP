Phylogenetic analysis

  To conduct phylogenetic analysis by integrating synteny information, we first used the method of constructing a phylogenetic tree and drew a tree like branching diagram using FastTree tool. Bootstrap is generally used to evaluate and test evolutionary trees, with repeated sampling values >1000. When the Bootstrap value of a node is greater than 70, the branch is considered reliable. Then, we integrated synteny information into the phylogenetic tree and observed the expansion pattern of the family gene in each genome after recursive WGD.
